# DocGenie

DocGenie turns your written notes into a clean, professional documentation website.
No subscriptions. No lock-in.

## Install
npm install -g docgenie

## Quick Start
docgenie init
docgenie build

## What It Is
- Static documentation generator
- Markdown in, HTML out
- You own the output

## What It Is Not
- Not a CMS
- Not an AI writer
- Not a hosted platform
